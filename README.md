## 📖 Tentang Saya

Assalamu'alaikum! Saya **Amelia**, mahasiswa Program Studi **Manajemen Bisnis Syariah (MBS)** di Fakultas Ekonomi dan Bisnis Islam (FEBI), Institut Agama Islam Negeri (IAIN) Palopo. Saya memiliki minat dalam pengembangan bisnis syariah, ekonomi digital, dan kewirausahaan berbasis nilai-nilai Islam.

Saat ini saya sedang:

- 🌱 Mempelajari konsep-konsep **manajemen bisnis syariah** dan implementasinya dalam dunia nyata.
- 🔭 Mengikuti program magang **Merdeka Belajar Kampus Merdeka (MBKM)** di BAZNAS Kota Palopo dan PT. Bumi Mineral Sulawesi (BMS) Luwu.
- 💬 Aktif dalam kegiatan **Himpunan Mahasiswa Program Studi (HMPS) MBS**, termasuk acara "Manajemen Legacy dan Sekolah Bisnis" untuk mahasiswa baru.
